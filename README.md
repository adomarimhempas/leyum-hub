# leyum-hub

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容涵盖不同主题，彼此无直接关联，适合作为静态页面集合进行统一管理。

## 项目简介

leyum-hub 是一个轻量级的静态页面存档仓库。主要用途包括：

- 保存独立完成的 HTML 页面作品
- 存放一些临时的、实验性的前端小页面
- 作为页面发布前的预览与测试空间

仓库本身不绑定任何特定域名或服务，所有页面均以文件形式存在，可自由部署至支持静态页面的托管平台。

## 目录说明

仓库根目录下按页面主题或类型组织文件夹，例如：

```
/
├── page-a/          # 页面 A 相关文件
├── page-b/          # 页面 B 相关文件
└── README.md        # 本文件
```

每个子文件夹通常包含一个独立的 HTML 文件及其所需的资源（如 CSS、JavaScript、图片等）。文件夹命名尽量采用简短、有意义的英文单词或缩写。

## 页面归档说明

- 每个页面均为自包含的 HTML 文件，尽可能减少外部依赖。
- 页面中不包含跟踪代码、广告或任何形式的第三方主动加载内容。
- 部分页面可能引用公共 CDN 资源（如字体、图标库），这些引用会在页面中明确标注。
- 归档页面仅供展示或参考，不保证长期可用性。

## 维护说明

本仓库由个人维护，更新频率不定。以下情况可能导致页面被更新或移除：

- 页面内容存在明显错误或过时信息
- 页面资源无法正常加载
- 仓库结构需要调整

欢迎提交 Issue 反馈问题，但请注意本仓库不提供技术支持或内容定制服务。
